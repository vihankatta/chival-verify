# Support

**Something broken or confusing?** Open a [GitHub Issue](https://github.com/vihankatta/chival-verify/issues) — bug reports and questions are both welcome there. Use the bug report template if you have a reproducible failure; otherwise just describe what you expected vs. what happened.

**Not sure if it's a bug or how to use something?** Open an issue anyway. There isn't a separate high-traffic community forum for this project yet, so issues are genuinely the fastest way to reach the maintainer.

**Want the full platform (live model connections, regression history, CI-ready reports)?** That's [Chival](https://chival.ai) — email partner@chival.ai.

**Response time:** this is a solo-maintained project. Real issues get a real response within about 48 hours (see [CONTRIBUTING.md](CONTRIBUTING.md)).
