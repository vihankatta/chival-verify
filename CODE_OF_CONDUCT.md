# Code of Conduct

## Our standard

Be direct, be kind, assume good faith. Technical disagreement is expected and welcome; personal attacks, harassment, or discrimination are not.

Examples of behavior that helps this project:

- Giving and receiving specific, honest technical feedback
- Focusing critique on the code or the idea, not the person
- Acknowledging when you're wrong, and saying so plainly

Examples of unacceptable behavior:

- Harassment, insults, or personal attacks, public or private
- Publishing others' private information without consent
- Any conduct that would reasonably make someone feel unwelcome or unsafe

## Scope

This applies in all project spaces — issues, pull requests, discussions — and in any other setting where someone is representing the project.

## Enforcement

Report unacceptable behavior to partner@chival.ai. Every report will be read. Consequences for violations range from a warning to a permanent ban from project spaces, at the maintainer's discretion, scaled to severity.

This is a small, solo-maintained project today. "Enforcement" realistically means one person reading your report and making a judgment call — said plainly rather than implying a process that doesn't exist yet.
